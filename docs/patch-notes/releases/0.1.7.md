## 이번 업데이트

Windows에서도 프로젝트 경로와 채팅 목록을 덜 헷갈리게 다듬었습니다.

### 주요 변경
- Windows 경로 앞의 불필요한 접두사가 화면에 보이지 않도록 했습니다.
- **프로젝트 경로 변경**은 폴더 선택 창에서 바로 고를 수 있습니다.
- 세션 인덱스가 빠진 채팅도 복구 흐름에서 다시 이어갈 수 있게 했습니다.
- 자동 업데이트가 실패하면 직접 업데이트하는 방법을 안내합니다.

### 사용성 개선
- 화면을 내려도 **프로그램 종료** 버튼이 상단에 남아 있습니다.
- 백업과 채팅 목록에서 긴 경로가 버튼을 밀어내지 않도록 했습니다.
- 패치노트의 점선 테두리가 스크롤 중 본문을 가리지 않게 했습니다.
