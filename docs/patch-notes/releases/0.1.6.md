## 이번 업데이트

다른 컴퓨터로 채팅을 옮길 수 있도록 채팅 내보내기/가져오기 흐름을 추가했습니다.

### 주요 변경
- 선택한 채팅을 `.tgz` 압축 파일로 내보낼 수 있습니다.
- 다른 mac에서 내보낸 채팅 파일을 가져와 대화를 이어갈 수 있도록 세션 파일, 인덱스, DB 참조를 함께 복원합니다.
- 내보내기 완료 모달에서 바로 백업 폴더를 열 수 있습니다.

### 사용성 개선
- 기존 **채팅 백업** 문구를 **채팅 내보내기**로 바꿔 백업 목록과 헷갈리지 않게 했습니다.
- 채팅 내보내기/가져오기 산출물은 백업 목록에 표시하지 않도록 정리했습니다.
- 내보내기 파일명을 짧고 알아보기 쉬운 형식으로 바꿨습니다.
- 업데이트 영역의 보조 설명 문구를 제거했습니다.

### 내부 정리
- mac 간 이전 경로와 archived session 내보내기 처리를 점검했습니다.
- Windows 경로 형태를 더 넓게 인식하도록 방어 로직을 보강했습니다.
- 채팅 전송 기능 주변의 죽은 분기와 미사용 연결을 정리했습니다.
