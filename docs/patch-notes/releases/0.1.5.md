## 이번 업데이트

채팅 상태 인식과 업데이트 확인 화면을 정리했습니다.

### 정리한 것
- **Codex DB 인식**을 보강해 최신 세션 상태를 더 정확히 읽도록 했습니다.
- 채팅과 프로젝트가 잘못된 상태로 보이던 문제를 실제 경로 기준으로 정리했습니다.
- 프론트엔드와 서버 코드를 역할별 파일로 나눠 이후 수정 범위를 줄였습니다.

### 사용성 개선
- **Codex 연결 상태**와 **업데이트 확인** 영역을 사이드 패널에서 더 찾기 쉽게 정리했습니다.
- 패치노트 모달에서 최신 업데이트를 강조하고, 폭과 글꼴을 읽기 좋게 다듬었습니다.
- 업데이트 상태 문구에 현재 버전과 업데이트 버전이 더 잘 보이도록 했습니다.

### 내부 정리
- 회귀 테스트와 릴리스 점검 흐름을 추가했습니다.
